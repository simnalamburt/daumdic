daumdic-ruby
========
[Daum Dictionary] API written in Ruby.

```ruby
require 'daumdic'

Daumdic.one_liner('elope')
# => "[ilóup]  눈이 맞아 함께 달아나다, 가출하다, 도망가다"
```
