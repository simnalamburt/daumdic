daumdic-ruby [![rubygems-i][]][rubygems-a]
========
[Daum Dictionary][] API written in **ruby**

```ruby
require 'daumdic'

Daumdic.search('elope')
# => "[ilóup]  눈이 맞아 함께 달아나다, 가출하다, 도망가다"
```
